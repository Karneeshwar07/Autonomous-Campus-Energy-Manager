# Autonomous Campus Energy Manager

An AI-powered dashboard built with **Python** and **Streamlit** to manage classroom energy usage.  
This project was originally developed for a hackathon and demonstrates how autonomous agents can optimize energy decisions.

---

## 🚀 Features
- ✅ ON/OFF decisions highlighted in green/red
- 📊 Usage charts showing per-classroom trends
- 🔄 Threshold updates tracking system adaptation
- 🎚️ Interactive slider to test decisions live
- 📥 Download button for logs

---
